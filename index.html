What is this?

This is AI Profile Picture Maker and i made this with object detection technology using Tensorflow.js and COCO-SSD models.

Object detection is a computer technology related to computer vision and image processing that deals with detecting instances of semantic objects of a certain class (such as humans, buildings, or cars) in digital images and videos. more info

TensorFlow.js is a library for machine learning in JavaScript.
COCO is a large-scale object detection, segmentation, and captioning dataset.
